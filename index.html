Le Front-end:

  • Gestion de projet, méthode AGILE.

Le Back-end

  • Création de base de données (Mysql)
  • PHP pour accéder à la bdd.
  • Côté framework Back-end:
    • PHP : Symfony - Laravel - CodeIgniter
